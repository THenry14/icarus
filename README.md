icarus
======
## Prerequirements
* node with npm
* grunt-cli
* mongodb

## Installation
```shell
npm install
```
Sometimes might be necessary to export env variable
```shell
PYTHON=python2 npm install
```

## Building
```shell
grunt build
```

## Running server
Currently it does read from **public** dir
```shell
npm start
```

## Runing tests
```shell
grunt test
```
